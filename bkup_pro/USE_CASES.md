## üè´ School & Education

### Example 1: Inter-Class Cricket Tournament
**Scenario**: 6 classes competing in a cricket tournament

**Setup**:
- Template: Gully Cricket (modify team names)
- Teams: Class 9A, Class 9B, Class 10A, Class 10B, Class 11A, Class 11B
- Format: Round Robin
- Duration: 1 week (one match per day)

**Workflow**:
1. Load "Gully Cricket" template
2. Edit team names to class names
3. Generate schedule
4. Score one match each day during lunch break
5. View standings to build excitement
6. Top 2 teams play finals
